安装

由于工作环境选择了Windows，所以需要进行一点修改和配置。首先Hugo是不支持cmd和Windows Powershell的，所以以下的所有内容都将在Git bash或Powershell下执行。

winget install Hugo.Hugo.Extended

然后通过 hugo new site hugo-blog 新建站点。

配置

以下是原本的目录结构：

my-site/
├── archetypes/
│   └── default.md
├── assets/ 
├── content/ 
├── data/ 
├── i18n/ 
├── layouts/ 
├── static/ 
├── themes/ 
└── hugo.toml <-- site configuration

由于我要用Obsidian管理文章，所以需要在hugo.toml里进行配置，将content文件夹替代为我的Obsidian可见的文件夹。

[[module.mounts]]
    source = 'C:\\Users\\TRSWNCA\\obsidian-files\\hugo'
    target = 'content'

这样就能在我的obsidian文件夹obsidian-files中进行文章的编写了。

主题

为了方便主题的自定义和修改，我首先将GitHub上的主题Fork到自己仓库下了，然后将其放在 themes 下：

git clone git@github.com:TRSWNCA/hugo-theme-diary.git themes/diary

然后在hugo.toml里修改theme的值为 diary。
